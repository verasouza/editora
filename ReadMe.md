## Desafio Xperian

### Objetivo 
 
Controle de estoque de uma editora.

**Regras**

- Cada livro deve estar associado a um Autor(a), mas nem todo Autor(a) precisa ter um livro;


**Tecnologias**

- Java 17;
- Banco de dados em memória(H2);
- Spring 3;
- Swagger;
- Actuator;

**Iniciar aplicação**

`` ./build.sh``
